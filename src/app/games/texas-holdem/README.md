# Texas Hold'em (单机德州扑克)

## Introduction (简介)
This is a single-player Texas Hold'em game built with Next.js and React.
这是一个使用 Next.js 和 React 构建的单机德州扑克游戏。

## Features (功能)
- **Single Player vs AI**: Play against 5 AI opponents. (单机对战 5 个 AI 对手)
- **Responsive Design**: Works on desktop and mobile. (响应式设计)
- **Fluid Animations**: Smooth card dealing and chip movements. (流畅的动画)
- **Game Logic**: Complete Texas Hold'em rules implementation. (完整的德州扑克规则实现)
